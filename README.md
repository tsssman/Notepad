Инструкция: В android studio нужно привязать свой github, затем нажимаем на 4 полоски вверху и переходим по 
Git -> Clone -> Url пишем https://github.com/tsssman/Notepad.git, указываем путь проекта и нажимаем на clone. 
Далее создаем виртуальное устройство (android studio предложит само), 
переходим по пути в проекте app -> kotlin + java -> com.example.flashcard выбираем файл MainActivity.kt и нажимаем вверху зеленую стрелочку для запуска, 
дальше android studio все сделает само
